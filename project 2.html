PROJECT 2: UX CASE STUDY

WEBSITE: JUMIA

OVERVIEW

Role: UI/UX Designer

Project Type: Concept Redesign of Landing Page

Tools Used: Figma, ChatGPT

This project focuses on redesigning the landing page of the Jumia website to improve usability, visual clarity, and trust for first-time and returning users. The redesign aims to simplify content presentation while maintaining Jumia’s brand identity as Africa’s leading online marketplace.

1.1 PROBLEM STATEMENT

Despite Jumia’s popularity, the existing landing page presents several usability challenges. Users are often greeted with a visually crowded interface filled with multiple promotions competing for attention. This makes it difficult for shoppers to quickly understand the value proposition, identify trusted brand stores, and navigate toward deals or key categories. As a result users may feel overwhelmed, leading to slower decision making and reduce engagement.

1.1.1 SOLUTION

Redesign the landing page of the website that shows simplicity and visual-clarity.

1.2 GOALS AND OBJECTIVES

The main goal of this redesign was to create a cleaner, more intuitive landing page experience that supports quick discovery and builds user trust.

Objectives:

  1. Simplify the landing page layout and reduce visual clutter.
  2. Improve content hierarchy and readability.
  3. Highlight official brand stores and deals more clearly.
  4. Encourage faster browsing and purchasing decisions.

1.3 RESEARCH AND KEY INSIGHTS

To understand user expectations, I had a one on one interactive sessions with a couple of my friends that had previously shopped on Jumia. I also took the initiative to review similar e- commerce platforms such as Amazon and Konga and observed common UX patterns used on effective landing pages.

Key Insights:

  1. Users value fast delivery, authentic products, and trusted sellers.
  2. Clear deals and discounts strongly influence buying decisions.
  3. Overloaded layouts reduce confidence and increase bounce rates.
  4. Trust signals such as reviews, official stores, and secure payment messaging are critical for e-commerce platforms.

1.4 KEY PROBLEMS

From the research and observations, the following problems were identified:

  1. Poor visual hierarchy makes it hard to scan the page
  2. Official brand stores are not clearly differentiated.
  3. Deals are visible but not well-structured.
  4. Testimonials and trust indicators lack prominence
